# Website with quiz generator for learning French
This web application is a local Flask-based trainer for French language learning, offering five interactive exercise types (True/False, Matching, Word Order, Fill blanks, Multiple Choice) to reinforce vocabulary and grammar. Users can upload text files to later generate exercises, with unlimited answer attempts and immediate visual feedback.
## Installation
```commandline
pip install -r requirements.txt
```
## Usage
```commandline
cd C:\path\to\exercise_generator\src\web    # for Windows
cd /path/to/exercise_generator/src/web      # for macOS/Linux
```
 ```commandline
python app.py
```
